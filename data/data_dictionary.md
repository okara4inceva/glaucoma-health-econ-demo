# Data Dictionary

The dataset `glaucoma_demo_patients.csv` is simulated and does not contain real patient data.

| Variable | Description |
|---|---|
| patient_id | Simulated ID |
| age | Simulated patient age |
| gender | Simulated gender |
| family_history_glaucoma | Family history risk factor |
| diabetes | Diabetes status |
| hypertension | Hypertension status |
| risk_score | Simple simulated risk score |
| risk_group | Low, Medium, or High |
| iop_mmhg | Simulated intraocular pressure |
| oct_result | Normal, Suspicious, or Abnormal |
| visual_field_result | Normal, Borderline, Mild defect, Moderate defect, or Severe defect |
| diagnosis | Confirmed glaucoma or no confirmed glaucoma |
| disease_stage | No glaucoma, Early, Moderate, or Advanced |
| management_plan | Simplified management category |
| annual_direct_cost_aed | Simulated annual direct medical cost |
| estimated_productivity_loss_aed | Simulated productivity loss |
| data_type | Label confirming simulated data |
